## גלריית השראה אינטראקטיבית מ-@julis.social

עמוד שמושך את התמונות מהאינסטגרם של julis.social דרך Firecrawl ומציג אותן בקנבס מוּדבורד חופשי — עם אפשרות לייצא הכל כתמונה/PDF.

### זרימת המשתמש
1. נכנסים לעמוד הבית — רואים את הקנבס נטען עם תמונות מהאינסטגרם של julis.social
2. אפשר לגרור, לסובב, ולשנות גודל של כל תמונה על הקנבס
3. כפתור "Shuffle" שמסדר מחדש את התמונות בצורה אקראית על הקנבס
4. אפשר להוריד את הקנבס כתמונה (PNG) או PDF

### מה נבנה

**1. חיבור Firecrawl**
- חיבור Firecrawl connector לפרויקט
- Server function שסורק את עמוד האינסטגרם של julis.social ומחלץ את כתובות התמונות

**2. עמוד הקנבס (דף הבית)**
- קנבס חופשי בסגנון מוּדבורד — תמונות בגדלים שונים מפוזרות עם רוטציה קלה
- גרירת תמונות (drag & drop) לשינוי מיקום
- אנימציות עדינות בטעינה
- עיצוב נקי עם צבעי רקע בהירים/ניטרליים

**3. כפתורי פעולה**
- "Refresh" — מושך תמונות חדשות מהאינסטגרם
- "Shuffle" — מסדר מחדש את התמונות בצורה אקראית
- "Download PNG" — מייצא את הקנבס כתמונה
- "Download PDF" — מייצא את הקנבס כ-PDF

**4. עיצוב**
- סגנון קולאז׳ חופשי עם צללים עדינים, רוטציות קלות, ומרחקים משתנים
- כותרת "@julis.social" בחלק העליון
- רספונסיבי למובייל ודסקטופ
